## 1. Компіляція та запуск Java:
- Встановіть JDK 21.
- Скомпілюйте файл: ```javac Lab1Variant12.java```.
- Запустіть програму: ```java Lab1Variant12```.
- Після завершення у папці ```results``` з’явиться файл ```res.csv```.
## 2. Побудова графіків:
- Встановіть Python
- Встановіть необхідні бібліотеки Python: ```pip install pandas matplotlib```.
- Запустіть скрипт: ```python ui_builder.py```.
