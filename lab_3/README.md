## 1. Компіляція та запуск Java:
- Встановіть JDK 21.
- Перейдіть в директорію `lab_3`.
- Скомпілюйте файли `.java`: ```javac -d ../ $(find . -name "*.java")```.
- Запустіть сервер: ```java -cp .. lab_3.Server```.
- Запустіть клієнт: ```java -cp .. lab_3.Client```.

## 2. Запуск Python клієнта:
- Переконайтеся, що встановлено Python версії 3.x.
- Перейдіть в директорію `lab_3`.
- Переконайтеся, що Java сервер вже запущений та працює.
- Запустіть клієнт: ```python3 client.py```.
